# 🌐 Personal Portfolio Website

A modern and interactive developer portfolio showcasing my **projects, skills, and experience**.  
Built with **React**, **Tailwind CSS**, and **Framer Motion** for smooth animations — connected to an **Express.js backend** with a **PostgreSQL/MySQL** database to handle contact form submissions securely.

---

## 🚀 Features

- 🎨 **Modern UI** with Glassmorphism design
- 💡 **Fully responsive** across all devices
- ⚡ **Smooth animations** powered by Framer Motion
- 📨 **Contact form** with backend integration (Express.js)
- 🗄️ **Database support** (PostgreSQL) for message storage
- ✉️ **Email notification system** for new contact submissions
- 🌍 **SEO-friendly** and optimized for fast loading

---

## 🛠️ Tech Stack

### **Frontend**
- React.js
- Tailwind CSS
- Framer Motion
- Axios (for API requests)

### **Backend**
- Node.js + Express.js
- PostgreSQL
- Nodemailer (for emails)
- Dotenv (for environment configuration)

# 🎨 Portfolio Frontend

This is the **React.js frontend** for my personal portfolio website.  
It provides a responsive, modern, and animated user interface that displays my projects, skills, and a contact form connected to the backend API.

---

## 🚀 Tech Stack

- **React.js (Vite)** – Frontend framework  
- **Tailwind CSS** – Styling  
- **Framer Motion** – Animations  
- **Axios** – API requests  
- **Vercel** – Deployment
